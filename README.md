In this repository i have used stanford dog data from kaggle which contains 120 types of dog breed. The split.py script decomposes the data into train data, test data, and validation data with proportion of 70%, 15% and 15%
respectively. Base architecture of EfficientNetB3 is used, with additional 3 layers including output layer using softmax activation fucntion. 
This project is done with help of claude and relvant sources in youtube to create a data repository and use prevous model on top of custom layers to understand more about deep learning techniques.
